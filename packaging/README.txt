Gauge — Claude Code / Codex 사용량 트레이 앱
=============================================

[ 실행 방법 ]
1. 이 폴더의 압축을 "옮기지 않을 위치"에 풉니다.
   (자동 시작을 켜면 이 경로를 기억하므로, 푼 뒤 폴더를 이동하지 마세요.)
2. Gauge.exe 를 더블클릭합니다.
3. 처음 실행 시 Windows SmartScreen "알 수 없는 게시자" 경고가 뜨면
   [추가 정보] → [실행]을 누르세요. (코드 서명을 하지 않은 정상 동작입니다.)

앱은 창이 없고 시스템 트레이(작업표시줄 오른쪽) 아이콘으로만 동작합니다.
트레이 아이콘을 왼쪽 클릭하면 사용량 팝오버가 열립니다.

[ 사용 전제 조건 ]
- Windows 10(빌드 19041+) 또는 Windows 11, 64비트(x64) PC.
  ※ ARM 전용 PC는 지원하지 않습니다(x64 에뮬레이션으로는 동작할 수 있음).
- Claude Code CLI / Codex CLI 가 설치되어 있고 "로그인"되어 있어야 합니다.
  로그인이 안 되어 있으면 해당 카드는 "데이터 없음"으로 표시됩니다.
- 별도의 .NET / Windows App SDK 설치는 필요 없습니다(앱에 포함되어 있음).

[ 동작 방식 / 개인정보 ]
- 각자 PC의 Claude Code / Codex 로그인 토큰 파일을 "읽기 전용"으로 읽어,
  사용량을 Anthropic / OpenAI "공식 엔드포인트"에서 직접 조회합니다.
- 토큰은 본인 PC를 떠나지 않으며, 제3자 서버로 전송되지 않습니다.
- 앱 안에는 어떤 자격증명도 들어있지 않습니다.

[ 종료 ]
- 트레이 아이콘 우클릭 → "종료".
- 자동 시작 등록/해제도 우클릭 메뉴에서 할 수 있습니다.
